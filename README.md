# 📱 My Telco – Gestion des Offres & Passes

## 📝 Introduction

Application permettant la gestion de forfaits et passes (souscription, résiliation, affichage, stockage local).

## ⚙️ Prérequis

- **Flutter SDK** : 3.29.1
- **Dart SDK** : 3.7.0
- Outils recommandés : VSCode
- (Optionnel) [FVM](https://fvm.app/) pour gérer les versions Flutter

Vérification de l’environnement :

```bash
flutter --version
dart --version
```
